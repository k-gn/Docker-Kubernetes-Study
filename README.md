# Docker-Kubernetes-Study Repository

## 인프런 도커 로드맵 스터디

### 초보를 위한 도커 안내서

> 실제로 느낀 도커의 장점
- git 처럼 어디서나 필요한 이미지를 pull 하면 개발 환경을 그대로 불러올 수 있다는 점
- 직접 험난한 서버 구축을 하지 않고도 세팅된 이미지를 통해 쉽게 서버를 구축할 수 있다는 점
- 도커 컨테이너를 여러개 띄운다 -> 모듈화 (자연스럽게 마이크로 서비스 구조 형태를 갖는다.)
- 결론 : 그냥 개쩐다.
